    Script setup:
    ```typescript
    <script setup lang="ts">
    import { computed } from 'vue'
    import type { ExhibitFindingEntry } from '@/data/exhibits'

    const props = defineProps<{
      findings: ExhibitFindingEntry[]
      heading?: string
    }>()

    const displayHeading = computed(() => props.heading ?? 'Findings')

    const columnPattern = computed(() => {
      if (props.findings.some(f => f.severity)) return 'severity'
      if (props.findings.some(f => f.background)) return 'background'
      return 'default'
    })
    </script>
    ```

    Template — `<section class="findings-table-section">`:

    1. Heading: `<h3 class="findings-table-heading">{{ displayHeading }}</h3>`

    2. Desktop table (`<table class="findings-table-desktop">`):
       - `<thead>` with dynamic headers based on columnPattern:
         - 'default': Finding, Description
         - 'severity': Finding, Description, Severity
         - 'background': Finding, Background, Resolution
       - `<tbody>` with `v-for` rows, `<td>` cells matching headers
       - Severity column cells: render `<span class="findings-table-badge findings-table-badge--{severity.toLowerCase()}">{{ finding.severity }}</span>` when severity exists

    3. Mobile cards (`<div class="findings-table-mobile">`):
       - `v-for` over findings, each `<div class="findings-table-card">`
       - Each card has labeled fields matching the columnPattern
       - Labels use `<span class="findings-table-label">` with field name
       - Values use `<span class="findings-table-value">`
       - Severity field in card: same badge span as desktop
