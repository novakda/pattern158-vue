Modify `src/App.vue` to conditionally mount FeedbackCollector using `defineAsyncComponent` with build-time gating.

The updated `<script setup>` block should be:
```typescript
import { defineAsyncComponent } from 'vue'
import NavBar from '@/components/NavBar.vue'
import FooterBar from '@/components/FooterBar.vue'

const FeedbackCollector = import.meta.env.MODE !== 'production'
  ? defineAsyncComponent(() => import('@/components/feedback/FeedbackCollector.vue'))
  : null
```

Add to template, after `<FooterBar />`:
```vue
<FeedbackCollector v-if="FeedbackCollector" />
```

The `import.meta.env.MODE !== 'production'` check is a compile-time constant that Vite replaces. In production builds, the condition is `false`, the dynamic import is unreachable, and the entire FeedbackCollector chunk (including feedback.css, composables, and all future feedback components) is tree-shaken. Zero bytes in production.
