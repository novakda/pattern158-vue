**Mapping rules per column structure:**

**Pattern 2 — Name/Title/Role (E, J):**
Each row `[col0, col1, col2]` maps to:
```typescript
{ name: col0, title: col1, role: col2 }
```

**Pattern 3 — Role/Involvement (L):**
The Role column contains embedded names separated by ` \u2014 ` (em-dash). Parse:
- Row `['Dan Novak \u2014 Development Lead', 'Brought in to build...']` becomes:
  `{ name: 'Dan Novak', title: 'Development Lead', role: 'Brought in to build...', isSelf: true }`
- Row `['External Power Platform Consultancy', 'Original architects...']` — no em-dash, treat entire Role value as title:
  `{ title: 'External Power Platform Consultancy', role: 'Original architects...' }`
- Row `['Lead Developer (anonymized)', 'Primary developer...']` — no em-dash, anonymous:
  `{ title: 'Lead Developer (anonymized)', role: 'Primary developer...' }`
- Row `['Governance Architect (anonymized)', 'Recipient of...']` — no em-dash, anonymous:
  `{ title: 'Governance Architect (anonymized)', role: 'Recipient of...' }`
