**src/components/FindingCard.vue:**

Single `finding: Finding` object prop (Pattern 3 from RESEARCH.md). Renders the NTSB-style article card. Uses TechTags for the tag list (passing `finding.tags` as `string[]` — the extended TechTags normalizes them). Conditionally renders the exhibit link if `finding.link` exists.

```vue
<script setup lang="ts">
import type { Finding } from '@/data/findings'
import TechTags from '@/components/TechTags.vue'

defineProps<{
  finding: Finding
}>()
</script>

<template>
  <article class="finding-card">
    <div class="finding-header">
      <span class="finding-label">&#x26A0; FINDING {{ finding.number }}</span>
      <h3 class="finding-title">{{ finding.title }}</h3>
    </div>
    <p class="finding-meta">{{ finding.meta }}</p>

    <div class="finding-section">
      <h4>Analysis</h4>
      <p>{{ finding.analysis }}</p>
    </div>

    <div class="finding-section">
      <h4>Solution</h4>
      <p>{{ finding.solution }}</p>
    </div>

    <div class="finding-section">
      <h4>Outcome</h4>
      <p>
        {{ finding.outcome }}
        <router-link v-if="finding.link" :to="finding.link" class="link-primary">
          Full investigation report &#x2192;
        </router-link>
      </p>
    </div>

    <TechTags :tags="finding.tags" />
  </article>
</template>
```

Note: Use `&#x26A0;` for the warning triangle (⚠) and `&#x2192;` for the right arrow (→) in the template — these are valid HTML character references that Vue templates handle correctly (unlike `&mdash;` which is an HTML5 named entity, numeric references like `&#x26A0;` work in Vue templates).

**src/components/InfluencesList.vue:**

Renders a `<dl>` definition list. Each influence maps to a `<dt>` + `<dd>` pair. The `<dd>` renders segments in order — each segment is either plain text or a text+link pair. Uses `<router-link>` for links.

```vue
<script setup lang="ts">
import type { Influence } from '@/data/influences'

defineProps<{
  influences: Influence[]
}>()
</script>

<template>
  <section class="content-section influences-compact">
    <div class="container">
      <h2>Five Core Influences</h2>
      <p class="section-subtitle">
        Not abstract inspirations&#x2014;specific sources that shaped how I solve problems.
        <router-link to="/philosophy#influences" class="link-primary">Full details &#x2192;</router-link>
      </p>

      <dl class="influences-list">
        <template v-for="influence in influences" :key="influence.term">
          <dt>{{ influence.term }}</dt>
          <dd>
            <template v-for="(segment, i) in influence.segments" :key="i">
              <template v-if="segment.link">
                {{ segment.text }}<router-link :to="segment.link.to" class="link-primary">{{ segment.link.text }}</router-link>
              </template>
              <template v-else>{{ segment.text }}</template>
            </template>
          </dd>
        </template>
      </dl>
    </div>
  </section>
</template>
```
