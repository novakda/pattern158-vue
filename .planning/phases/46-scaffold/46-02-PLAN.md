Create `tsconfig.editorial.json` at the repo root with the JSON below. This is a verbatim mirror of `tsconfig.scripts.json` with three deltas:

1. `outDir`: `.tsbuildinfo-scripts` → `.tsbuildinfo-editorial`
2. `include`: `["scripts/markdown-export/**/*.ts"]` → `["scripts/editorial/**/*.ts"]`
3. (Everything else identical — same target, module, lib, strictness, composite flags.)

`paths: {}` is mandatory — this is what enforces SCAF-08's "no `@/` aliases" rule at the type-checker level. If a Phase 47+ author writes `import x from '@/...'`, the editorial project will fail to resolve it.

`composite: true` is mandatory — required for the project to be referenced from the root tsconfig.

`module: NodeNext` is mandatory — matches `tsconfig.scripts.json` and is why Plan 01's placeholder imports use explicit `.ts` extensions (`import type { Route } from './routes.ts'`).

**File: `tsconfig.editorial.json`**
```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "strict": true,
    "composite": true,
    "declaration": true,
    "emitDeclarationOnly": true,
    "outDir": ".tsbuildinfo-editorial",
    "rootDir": ".",
    "isolatedModules": true,
    "esModuleInterop": true,
    "resolveJsonModule": true,
    "skipLibCheck": true,
    "lib": ["ES2022"],
    "types": ["node"],
    "paths": {},
    "baseUrl": "."
  },
  "include": ["scripts/editorial/**/*.ts"]
}
```

Notes:
- `resolveJsonModule: true` is retained from the analog because Phase 47 (CAPT-01) loads `src/data/json/exhibits.json`. Even though Phase 47 will use `fs.readFile` + `JSON.parse` (NOT ESM JSON import assertion — explicit forbidden in CAPT-01 wording), keeping the flag matches the analog and costs nothing.
- `types: ["node"]` is retained because Phase 47+ uses `node:fs`, `node:path`, `node:url`, `process`, etc.
- DOM lib is intentionally NOT included now — Phase 49 will add it ONLY if Turndown's TS types require DOM globals at the type level (likely they do — but defer that decision to Phase 49 to keep this scaffold minimal).
