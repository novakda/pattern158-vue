DOM shape — `.exhibit-quote` (exhibit-a.html):
```html
<blockquote class="exhibit-quote">
  <p>I'd consider the last couple days a success...</p>
  <footer class="attribution">
    <span>Chief of Learning Services, Metrics, Processes &amp; Technology, Electric Boat</span>
    <!----><!-- or: <span class="role">in summary email to EB leadership</span> -->
  </footer>
</blockquote>
```

Field derivation rules (unified per-blockquote function):
- Detect shape via class: `.testimonial-quote` OR `.exhibit-quote`.
- `text`:
  - testimonial-quote → `.quote-text` textContent.
  - exhibit-quote → direct `<p>` child textContent (first one).
- `attribution`:
  - testimonial-quote → `<cite>` textContent; fallback `''`.
  - exhibit-quote → first `<span>` (NOT `<span class="role">`) in `<footer.attribution>`; fallback `''`.
- `role`:
  - testimonial-quote → `.quote-context` textContent; fallback `''`.
  - exhibit-quote → `<span class="role">` textContent; fallback `''`.
- `sourcePageLabel`: `opts?.sourcePageLabel ?? ''`.

Shape detection: iterate the unified selector `blockquote.testimonial-quote, blockquote.exhibit-quote` and branch on `element.classList.contains('testimonial-quote')`.

```typescript
// scripts/tiddlywiki/extractors/testimonials.ts
// EXTR-06 — testimonial extractor (testimonial-quote + exhibit-quote shapes).
// SCAF-08: no setTimeout, no Date.now(), no new Date(), no Promise.all.

import { parseHtml, type Testimonial } from './types.ts'

const BLOCKQUOTE_SELECTOR = 'blockquote.testimonial-quote, blockquote.exhibit-quote'
const TESTIMONIAL_CLASS = 'testimonial-quote'

function textOf(el: Element | null): string {
  return (el?.textContent ?? '').trim()
}

function firstNonRoleSpan(footer: Element): Element | null {
  const spans = footer.querySelectorAll('span')
  for (const span of Array.from(spans)) {
    if (!span.classList.contains('role')) return span
  }
  return null
}

function extractTestimonialQuote(bq: Element): { text: string; attribution: string; role: string } {
  return {
    text: textOf(bq.querySelector('.quote-text')),
    attribution: textOf(bq.querySelector('cite')),
    role: textOf(bq.querySelector('.quote-context')),
  }
}

function extractExhibitQuote(bq: Element): { text: string; attribution: string; role: string } {
  const text = textOf(bq.querySelector(':scope > p'))
  const footer = bq.querySelector('footer.attribution')
  const attribution = footer === null ? '' : textOf(firstNonRoleSpan(footer))
  const role = textOf(bq.querySelector('footer.attribution span.role'))
  return { text, attribution, role }
}

export function emitTestimonials(
  html: string,
  opts?: { sourcePageLabel?: string },
): readonly Testimonial[] {
  const doc = parseHtml(html)
  const blockquotes = doc.querySelectorAll(BLOCKQUOTE_SELECTOR)
  const sourcePageLabel = opts?.sourcePageLabel ?? ''
  const out: Testimonial[] = []
  blockquotes.forEach((bq) => {
    const extracted = bq.classList.contains(TESTIMONIAL_CLASS)
      ? extractTestimonialQuote(bq)
      : extractExhibitQuote(bq)
    out.push({
      text: extracted.text,
      attribution: extracted.attribution,
      role: extracted.role,
      sourcePageLabel,
    })
  })
  return out
}
```

Notes:
- Unified selector `blockquote.testimonial-quote, blockquote.exhibit-quote` — one DOM pass, branch per shape.
- `firstNonRoleSpan` uses `Array.from(spans)` → plain JS array → `for-of` iteration is safe (ES2022 native). Snapshots the NodeList.
- `:scope > p` in exhibit-quote restricts to DIRECT p child (not nested content).
- `footer === null` guard keeps flow-sensitive null narrowing — TypeScript won't flag the `textOf(firstNonRoleSpan(footer))` call site.
