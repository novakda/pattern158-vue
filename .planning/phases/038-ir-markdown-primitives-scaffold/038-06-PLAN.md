Ship six inline/simple-block primitive IR factories per PRIM-01 and D-09/D-10: `text`, `heading`, `paragraph`, `link`, `wikilink`, `caption`. Each is a pure factory that returns a DocNode or InlineSpan — ZERO rendering logic, ZERO escape logic. Primitives accept convenience argument shapes (strings for common cases, or pre-built InlineSpan[] arrays for composed content).

Wikilink rendering strategy (for D-09 — primitives have no rendering knowledge):
  The wikilink primitive MUST produce a LinkSpan whose `href` field starts with a sentinel
  string that the Phase 42 Obsidian renderer recognizes and rewrites to `[[target]]` or
  `[[target|display]]` syntax. The Phase 41 mono renderer falls back to rendering it as
  a regular markdown link. The sentinel MUST be a constant exported from wikilink.ts so
  tests and renderers can import it directly instead of hard-coding a magic string.

  Exported constant: `WIKILINK_HREF_PREFIX = 'wikilink://'`
  Example: `wikilink('Exhibit A')` → `{ kind: 'link', href: 'wikilink://Exhibit A', children: [text('Exhibit A')] }`
  Example: `wikilink('Exhibit A', 'Link Text')` → `{ kind: 'link', href: 'wikilink://Exhibit A', children: [text('Link Text')] }`

```ts
//
// Phase 38 primitive: HeadingNode factory.
// Accepts either a plain string (sugar for `[text(s)]`) or pre-built InlineSpan[].
// Level is constrained to HeadingLevel (1-6) at compile time.

import { text } from './text.js'

export function heading(
  level: HeadingLevel,
  content: string | readonly InlineSpan[],

}
```

```ts
// scripts/markdown-export/primitives/wikilink.ts
//
// Phase 38 primitive: wikilink factory.
//
// Produces a LinkSpan whose `href` is prefixed with a sentinel string that
// the Phase 42 Obsidian renderer recognizes and rewrites into `[[target]]` or
// `[[target|display]]` syntax. The Phase 41 monolithic renderer will fall
// back to rendering these as plain markdown anchors (via slug lookup).
//
// Primitives are escape-free (D-09) — the renderer calls escapeWikilinkTarget()
// on the target portion at render time, NOT here.

import type { InlineSpan, LinkSpan } from '../ir/types.js'
import { text } from './text.js'

export const WIKILINK_HREF_PREFIX = 'wikilink://'

export function wikilink(target: string, display?: string): LinkSpan {
  const displayText = display ?? target
  const children: readonly InlineSpan[] = [text(displayText)]
  return {
    kind: 'link',
    href: `${WIKILINK_HREF_PREFIX}${target}`,
    children,
  }
}
```

```ts
// scripts/markdown-export/primitives/caption.ts
//
// Phase 38 primitive: image caption factory.

// Per VAULT-09 (Obsidian vault: images skipped, alt text emitted as italicized

import type { EmphasisSpan, ParagraphNode } from '../ir/types.js'

export function caption(captionText: string): ParagraphNode {

}
```
