Create `src/components/feedback/AnnotationCanvas.vue` as a Vue 3 SFC with `<script setup lang="ts">`:

**Props:**
- `screenshotSrc: string` — the base64 PNG data URI of the captured screenshot

**Exposed methods (via defineExpose):**
- `compositeScreenshot(): string` — returns a new PNG data URI with annotations composited onto the screenshot

**Internal state:**
- `activeTool: ref<'rect' | 'arrow'>('rect')` — current drawing tool
- `shapes: ref<Array<{ type: 'rect' | 'arrow'; startX: number; startY: number; endX: number; endY: number }>>([])` — committed shapes
- `drawing: ref<boolean>(false)` — whether user is currently dragging
- `currentShape: ref<{ type: 'rect' | 'arrow'; startX: number; startY: number; endX: number; endY: number } | null>(null)` — shape being drawn

**Template structure:**
```html
<div class="fb-annotation">
  <div class="fb-annotation-toolbar">
    <button :class="['fb-annotation-tool', { 'fb-annotation-tool--active': activeTool === 'rect' }]"
      @click="activeTool = 'rect'" title="Rectangle">&#9633;</button>
    <button :class="['fb-annotation-tool', { 'fb-annotation-tool--active': activeTool === 'arrow' }]"
      @click="activeTool = 'arrow'" title="Arrow">&#8599;</button>
    <button class="fb-annotation-tool" @click="undo" :disabled="shapes.length === 0" title="Undo">&#8630;</button>
  </div>
  <div class="fb-annotation-canvas-wrap">
    <img :src="screenshotSrc" class="fb-annotation-img" ref="imgRef"
      @load="onImageLoad" draggable="false" />
    <canvas ref="canvasRef" class="fb-annotation-canvas"
      @mousedown="onMouseDown" @mousemove="onMouseMove" @mouseup="onMouseUp"
      @mouseleave="onMouseUp" />
  </div>
</div>
```

**Drawing logic:**
- `onImageLoad`: set canvas width/height to match the `<img>` rendered dimensions (offsetWidth/offsetHeight). Store the natural-to-rendered scale ratio for compositing.
- `onMouseDown(e)`: set `drawing = true`, record startX/startY from `e.offsetX, e.offsetY`, create `currentShape` with `type: activeTool`.
- `onMouseMove(e)`: if `drawing && currentShape`, update `currentShape.endX/endY`, call `redraw()`.
- `onMouseUp(e)`: if `drawing && currentShape`, finalize endX/endY, push to `shapes` only if the shape has meaningful size (Math.abs(endX-startX) + Math.abs(endY-startY) > 5), set `drawing = false`, `currentShape = null`, call `redraw()`.
- `undo()`: `shapes.pop()`, call `redraw()`.

**redraw() function:**
- Clear entire canvas.
- For each shape in `shapes` plus `currentShape` (if not null):
  - Set `ctx.strokeStyle = '#ff3333'`, `ctx.lineWidth = 2`, `ctx.lineCap = 'round'`.
  - If `type === 'rect'`: `ctx.strokeRect(x, y, w, h)`.
  - If `type === 'arrow'`: draw a line from start to end, then draw an arrowhead at the end point. Arrowhead: two lines forming a 30-degree angle from the line direction, each 12px long. Use `Math.atan2(endY - startY, endX - startX)` to get the angle, then draw two lines at `angle + Math.PI * 5/6` and `angle - Math.PI * 5/6` from the end point.

**compositeScreenshot() method:**
- Create an offscreen canvas sized to the screenshot's natural dimensions (from the `<img>` naturalWidth/naturalHeight).
- Draw the screenshot image onto it: `ctx.drawImage(imgRef, 0, 0)`.
- Compute scale factor: `scaleX = naturalWidth / renderedWidth`, `scaleY = naturalHeight / renderedHeight`.
- For each shape in `shapes`, draw it on the offscreen canvas using scaled coordinates (`startX * scaleX`, etc.) and scaled lineWidth (`2 * Math.max(scaleX, scaleY)`). Use the same drawing logic as redraw() but with scaled values.
- Return `offscreenCanvas.toDataURL('image/png')`.
