    1. Create src/components/feedback/PickerOverlay.vue using `<script setup lang="ts">`:

    **Template structure:**
    - A full-viewport transparent div with class `fb-picker-overlay` and `pointer-events: all`
    - A highlight div with class `fb-picker-highlight` positioned absolutely, styled to show the outline around the hovered element. Bind its style to reactive position/size state.

    **Mousemove handler on the overlay div:**
    - On mousemove, temporarily set `pointer-events: none` on the overlay
    - Call `document.elementFromPoint(event.clientX, event.clientY)` to find the element underneath
    - Restore `pointer-events: all` on the overlay
    - If the found element has a class starting with `fb-` or is inside a `.fb-root` ancestor, ignore it (skip self-selection per CONTEXT.md decision)
    - Otherwise, update the highlight div position/size from the found element's `getBoundingClientRect()`
    - Store the found element as `hoveredElement` ref

    **Click handler on the overlay div:**
    - Prevent default and stop propagation
    - If `hoveredElement` is set, call `captureElement(hoveredElement)` from captureElement.ts
    - Call `feedback.selectElement(hoveredElement, capture)` to transition state to annotating

    **Keyboard handling (Escape):**
    - Register a keydown listener on mount via `onMounted`
    - On Escape key, call `feedback.deactivate()` to return to idle
    - Clean up listener in `onUnmounted`

    **Cursor:** Add `cursor: crosshair` to the overlay div.

    2. Register global Ctrl+Shift+F keyboard shortcut in FeedbackCollector (this component is always mounted in dev, making it the right place for document-level shortcuts):
       ```typescript
       import { onMounted, onUnmounted } from 'vue'

       function onKeyDown(e: KeyboardEvent) {
         if (e.ctrlKey && e.shiftKey && e.key === 'F') {
           e.preventDefault()
           if (feedback.state.phase === 'idle') {
             feedback.activate()
           } else if (feedback.state.phase === 'picking') {
             feedback.deactivate()
           }
         }
       }

       onMounted(() => document.addEventListener('keydown', onKeyDown))
       onUnmounted(() => document.removeEventListener('keydown', onKeyDown))
       ```

    3. Add PickerOverlay to template, conditionally rendered when phase is 'picking':
       ```html
       <Teleport to="body">
         <div class="fb-root">
           <FeedbackTrigger
             :disabled="!config.isConfigured"
             @activate="feedback.activate()"
           />
           <PickerOverlay v-if="feedback.state.phase === 'picking'" />
         </div>
       </Teleport>
       ```
