<decisions>
## Implementation Decisions

### EDIT-05 scaffold emission
- **Location in tool flow:** after the primary atomic write succeeds in `main()` (after `writePrimaryAndMirror` returns), BEFORE the summary print. Failures in scaffold emission are non-fatal тАФ logged to stderr, do not change exit code.
- **Path derivation:** `path.join(path.dirname(config.outputPath), 'site-editorial-findings.md')` тАФ sits next to the capture artifact in the vault.
- **Idempotency (never overwrite):** `fs.access(path, F_OK)` check тАФ if file exists, skip emission silently (no log). This is critical тАФ Dan's populated findings must survive subsequent tool runs.
- **Template content:** five sections per the SC list + 3 cross-reference stubs in a header block. Uses the locked section names: `Inconsistencies`, `Structural`, `Copy`, `Alignment Gaps`, `Open Questions`. Each section seeded with a prompt line describing what goes there + priority-label reminder.
- **Priority labels**: `blocker` / `should-fix` / `nice-to-have` (locked).
- **Cross-ref docs**: `career/reference-data/design-philosophy-essay.md`, `career-values-reference.md`, `case-study-gp-accessibility-signoff.md`.

### Export surface
- New helper `emitFindingsScaffold(outputPath)` in `scripts/editorial/write.ts` (or new `findings-scaffold.ts` module if cleaner). Pure-ish тАФ reads and writes fs; no other side effects.
- Called from `index.ts` `main()` after the primary write.

### SCAF-08 Discipline
- No `Date.now()`, `Math.random()`, `os.EOL`, `@/`, `Promise.all`. Literal `\n` throughout the template.

### Tests
- Unit test: scaffold written when file absent; scaffold NOT written when file present; template contains all 5 section headings + 3 cross-ref filenames + all 3 priority labels.
- Integration check via existing index.test.ts: happy-path test asserts scaffold write call is attempted after writePrimaryAndMirror.

</decisions>

- **Helper function shape:**
  ```ts
  export async function emitFindingsScaffold(captureOutputPath: string): Promise<string | null> {
    const scaffoldPath = path.join(path.dirname(captureOutputPath), 'site-editorial-findings.md')
    try {
      await fsp.access(scaffoldPath, constants.F_OK)
      return null  // already exists, do not overwrite
    } catch {
      await fsp.writeFile(scaffoldPath, SCAFFOLD_TEMPLATE, { encoding: 'utf8' })
      return scaffoldPath
    }
  }
  ```
